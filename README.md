# Interactive Solar System Model using **VPython**

## Dependencies:
 - Python 3.11
 - VPython

## Installation:

1. Clone the repository:
```bash
git clone https://github.com/Smarkk2367/Solar_System.git
```

2. Set up a virtual environment using python 3.11

3. Download the dependencies:
```python
pip install requirements.py
```

4. Run the project!
